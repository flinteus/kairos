
## Установка

1. Скачайте все файлы расширения в одну папку
2. Откройте Chrome и перейдите на страницу `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами расширения

## Использование

1. Нажмите на иконку расширения на панели инструментов Chrome
2. Настройте параметры пароля:
   - Длина (от 6 до 32 символов)
   - Типы символов: заглавные/строчные буквы, цифры, спецсимволы
   - Исключение похожих символов (i, l, 1, o, 0)
3. Нажмите "Сгенерировать пароль"
4. Используйте кнопки для копирования или генерации нового пароля
5. Следите за оценкой сложности пароля
6. Все сгенерированные пароли сохраняются в истории (до 10 последних)

## Функции

- Генерация случайных паролей с настраиваемыми параметрами
- Визуальная оценка сложности пароля
- Копирование пароля в буфер обмена одним кликом
- История сгенерированных паролей

## Безопасность

Все пароли генерируются локально в вашем браузере и не передаются на серверы.
История паролей хранится только в localStorage вашего браузера.
